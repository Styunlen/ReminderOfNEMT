## 安装教程

在Release中下载最新版本的压缩包，然后选择自己电脑的版本解压

不知道自己电脑是X86还是X64的小伙伴可右键计算机，单击属性，就可以看到自己计算机是32位的还是64位的了

## 开机启动

按住`WIN`+ `R`或打开任务管理器，选择启动，打开运行

输入Shell:StartUp,回车即可打开启动文件夹

将高考倒计时程序的快捷方式，注意是快捷方式拖入文件夹内，即可完成开机启动
